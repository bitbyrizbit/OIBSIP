# OIBSIP

AICTE Oasis Infobyte Summer Internship Programme — Python Programming track.

Each folder below is a complete, independently deployable project built for
one task in the internship. Full documentation, architecture notes, and
setup instructions live inside each project's own README.

---

## Completed

**[Python-Task4-WeatherApp](./Python-Task4-WeatherApp)** — Glass, a reading
of the sky. A full-stack weather application built as an instrument reading
rather than a typical dashboard. FastAPI, Next.js, Framer Motion.
[Live](https://oibsip-sand.vercel.app)

**[Python-Task5-ChatApp](./Python-Task5-ChatApp)** — Party Line, pick up,
someone's on the line. A real-time chat application built around 1900s
party-line telephone culture rather than a generic messaging clone.
FastAPI, WebSockets, Next.js, Framer Motion.
[Live](https://oibsip-2ac8.vercel.app)
